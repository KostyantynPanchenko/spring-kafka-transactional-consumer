### Guides
* Clone and run [Spring Kafka transactional producer](https://github.com/KostyantynPanchenko/spring-kafka-transactional-producer.git) project first
* Run this project. Observe in logs how message is consumer only after transaction on producer side is completed
* Explore `application.properties` for Kafka consumer configuration
